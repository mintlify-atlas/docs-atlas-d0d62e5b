## Tool conventions

### Authentication

- **Free tools** work without an API key or account
- **Paid tools** require an API key from [trystobo.com](https://trystobo.com)
- **Account tools** require an API key but don't consume credits

### Return format

All tools return JSON responses with structured data. Success responses include the requested data, while error responses include an `error` field with details.

### Caching

- `audit_site` results are cached for 24 hours
- Other audits are not cached and always run fresh
- Generators always create new content

<Note>
Free tools (audits and most generators) work without an API key. You can start using Stobo MCP immediately without creating an account.
</Note>
